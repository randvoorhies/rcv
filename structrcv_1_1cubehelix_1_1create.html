Detailed Description

A helper struct to create cubehelix instances using named parameters. 

          cubehelix colorizer = cubehelix::create().start(.6).rot(1.5).nlev(128).reverse();

          cv::Mat grayscale;
          cv::Mat colorized = colorizer(grayscale);
